# House Price Prediction

This project aims to predict apartment rental prices using machine learning algorithms and Artificial Neural Network (ANN).

## Project workflow:

### P1. Data Analysis and Preprocessing
- Explore and clean the dataset
- Handle missing values and outliers
- Perform feature selection and encoding
- Split data into training, validation and test sets
- Scale features using StandardScaler

### P2. Model Training and Prediction
Predict apartment prices using:
- K-Nearest Neighbors (KNN)
- Decision Tree (DT)
- Random Forest (RF)
- Artificial Neural Network (ANN) using TensorFlow/Keras

### P3. Hyperparameter Tuning
- Optimize hyperparameters for each model
- Evaluate and compare performance metrics (MAE, MAPE, RMSE, R²)
- Select the best-performing model

### P4. Neural Network Optimisation
- Implement three ANN architectures (Small, Medium, Large)
- Experiment with different layers, activation functions, and regularization strategies
- Apply EarlyStopping to prevent overfitting

## Dataset
The original dataset is available at the [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/555/apartment+for+rent+classified).

## Tools and Libraries
- Programming Language: Python
- Data Manipulation: Pandas, NumPy
- Data Visualisation: Matplotlib, Seaborn
- Machine Learning: Scikit-learn
- Deep Learning: TensorFlow / Keras
